# ENGXCONSOLE 엔진 소개

## 저자: Xesrs5

### _이 엔진은 아직 개발중이며 바뀔수있습니다._

> 엔진에 관한 도구와 웹사이트는 언제나 환영입니다.

> 모든 코드는 사용자가 직접 써야합니다.

LoopG는 계속 실행되고 StartG는 시작하고 처음으로 한번 실행됩니다.
WinName은 창 이름을 설정합니다. 기본값은 xesrs5입니다.
엔진에 꼭 필요한 헤더는 chain.h와 windows.h 헤더입니다.

> 엔진코드는 마음대로 하셔도되지만 오류가 있을수도 있습니다.

컴파일은 파일안에 있는 compile.bat를 실행하시면 됩니다.
 (추가 사항이 있다면 메모장으로 compile.bat 코드를 편집하세요.)

피드백 해주실게 있다면 Xesrs5@gmail.com 으로 메일을 보내주세요.

글씨 출력은 windows api를 이용하셔도 되고 stdio.h같은 헤더를 사용하여도 됩니다.

헤더와 코드를 직접 추가해서 엔진을 더 편리하고 기능을 추가할수 있지만 올린뒤 **ENGXCONSOLE에 사용할 코드입니다** 라고 명시 해주세요.

이 엔진에서 만든 게임은 팔아도 됩니다. (크레딧에 ENGXCONSOLE 엔진을 사용했다고 명시해 주세요 자세한것은 LICENSE 파일 참조.)

헤더와 코드를 만들어서 쓸 경우 chain.h에 연결하는걸 권장합니다.


변수 이름은 모두 편하도록 소문자로 해놨습니다

현재 엔진에 있는 기능:

---------------------------------------------
### engx_printc
- 콘솔에 글자를 출력할떄 사용합니다.
 - engx_printc("여기에 글자를 쓰면 됍니다.");
---------------------------------------------
### engx_loopg
- loopg를 끄거나 킬떄 사용합니다.
 -  키기 \ engx_loopg = 1; / 끄기 \ engx_loopg = 0;
---------------------------------------------
